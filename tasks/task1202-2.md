### 구현 완료 내역

#### 1. validateFiles.ts 유틸리티 파일 생성 ([src/utils/validateFiles.ts](src/utils/validateFiles.ts))
- 파일 포맷 검증 함수 (`validateFileFormat`, `validateFilesFormat`)
- 파일 개수 검증 함수 (`validateFileCount`)
- 종합 검증 함수 (`validateFiles`)
- 분석 버튼 활성화 확인 함수 (`canAnalyze`)
- 타입 정의: `ValidationResult` (isValid, errorType, message, messageType)

#### 2. UploadPage.jsx 리팩토링 ([src/components/UploadPage.jsx](src/components/UploadPage.jsx))
- validateFiles 유틸리티 함수 사용
- 상태 관리: `validationMessage`, `validationMessageType`로 변경
- 파일 선택 시 자동 검증 및 피드백
- 파일 삭제 시 자동 검증 및 피드백
- 메시지 타입별 스타일 적용 (error, warning, success, info)

#### 3. 메시지 타입별 피드백
- **FORMAT_ERROR**: 포맷 오류 시 에러 메시지 표시
- **MIN_COUNT_WARNING**: 20장 미만 시 경고 메시지
- **MAX_COUNT_EXCEEDED**: 30장 초과 시 에러 메시지
- **SUCCESS**: 30장 정확히 선택 시 성공 메시지
- **INFO**: 20-29장 선택 시 정보 메시지 (없음)
